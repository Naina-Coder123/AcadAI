# 📚 AcadAI - Multi-Agent System for Personalized Academic Learning & Assistance

---

## 🚀 Overview

**AcadAI** is an AI-powered academic assistant designed to provide **personalized, syllabus-aligned learning support** for students.

Unlike general AI systems, AcadAI combines:
- 🧠 Large Language Models (LLMs)
- 🔍 Retrieval-Augmented Generation (RAG)
- 🤖 Multi-Agent Architecture

to deliver **accurate, context-aware, and exam-oriented responses**.

---

## 🎯 Problem Statement

Most AI tools:
- Provide **generic answers**
- Ignore **college syllabus & notes**
- Lack **exam-oriented explanations**

📌 **AcadAI solves this by grounding responses in:**
- Syllabus documents
- Lecture notes
- Previous year papers
- Scanned PDFs

---

## 🎯 Objectives

- Build a **personalized academic assistant**
- Integrate **LLM + RAG**
- Implement a **multi-agent workflow**
- Support **PDF & OCR-based learning**
- Reduce hallucination using **retrieval grounding**
- Deliver **step-by-step explanations**

---

## ✨ Key Features

- 📖 Syllabus-based answers
- 🤖 Multi-agent pipeline (Router, Reasoning, Tutor, Critic)
- 🔍 Context-aware retrieval (RAG)
- 📄 OCR support for scanned documents
- 🧠 Structured explanations
- 📝 Notes summarization & Q&A
- ⚡ Improved accuracy & reduced hallucination
